====================
Semagia Oomap Loomap
====================

GUI for Topic Maps <http://www.isotopicmaps.org/> query languages.

Supported languages:
  * TMQL <http://www.isotopicmaps.org/tmql/>
  * tolog <http://www.ontopia.net/topicmaps/materials/tolog.html>
  * Toma <https://ontopia.googlecode.com/svn/trunk/sandbox/toma/doc/TW_UM_TOMA_110.pdf>


Similar projects:
  * TMQL Console <https://github.com/mhoyer/tmql-console/tree/master/tmql-console>
  * Tamana <https://code.google.com/a/eclipselabs.org/p/tamana/>
